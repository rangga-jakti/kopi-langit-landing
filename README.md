# Kopi Langit — Landing Page

Landing page komersial untuk kedai kopi "Kopi Langit" di Bandung.

## Stack
- Next.js 14 + TypeScript + Tailwind CSS + Framer Motion

## Cara Menjalankan
```bash
npm install
npm run dev      # localhost:3000
npm run build    # output: dist/
```

## Catatan
- Ganti nomor WA: cari `6281234567890` di semua file
- Taruh foto di `public/images/`
- Embed Google Maps di `Location.tsx`
